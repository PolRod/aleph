ALEPH

Aleph is a Redshift data analysis application. At its most basic, users write SQL and get results. On top of that we have build a rich suite of features which focus on developing and collecting data investigation techniques, serving as a repository of institutional knowledge.

SQL Authoring Tools

Table schema -- schema, table, and column names -- is analyzed and fed into SQL editor autocomplete along with Redshift SQL syntax and keywords. Parameters and SQL snippets allow expressive templatizated queries, allowing expert users to create frameworks for others.

Track and Share Analyses

Query versions are maintained in Git, leveraging the Github UI to document and history of SQL changes. Comments allow people to leave notes to each other.

Manage Access

Access control granular on the table level allow teams to do research in their own domain, ensuring that sensitive information is protected
